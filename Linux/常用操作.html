netstat ​

netstat 是一个常用的网络工具，可以显示网络连接、路由表、接口统计信息等。

bash
sudo netstat -uln

    -t: 显示 TCP 端口
    -u: 显示 UDP 端口
    -l: 显示监听状态的端口
    -n: 以数字形式显示地址和端口号，而不是解析成域名和服务名

vmware 挂载共享文件夹 ​

在 VMware 虚拟机中挂载共享文件夹通常用于在主机和虚拟机之间共享文件。以下是如何在 VMware 虚拟机中挂载共享文件夹的步骤：

1. 在 VMware 中设置共享文件夹 ​

  1. 启动 VMware 虚拟机：确保你的虚拟机正在运行。
  2. 打开虚拟机设置：
    • 在 VMware 窗口中，选择你的虚拟机。
    • 点击菜单栏中的 虚拟机 -> 设置。
  3. 添加共享文件夹：
    • 在设置窗口中，选择 选项 标签。
    • 选择 共享文件夹，然后点击右侧的 添加 按钮。
    • 在弹出的向导中，点击 下一步。
    • 选择你要共享的文件夹路径，并为其命名。
    • 选择共享文件夹的访问方式（例如，启用或禁用只读访问）。
    • 点击 完成 完成设置。

2. 在虚拟机中挂载共享文件夹 ​

vmhgfs-fuse: 这是 VMware 提供的一个工具，用于在虚拟机中挂载主机共享文件夹。vmhgfs-fuse 是基于 FUSE（Filesystem in Userspace）的实现，允许用户空间程序实现文件系统功能。

.host:/: 这是 VMware 虚拟机中表示主机共享文件夹的特殊路径。.host:/ 表示主机的根目录，通常用于挂载主机上的共享文件夹。

/mnt/hgfs: 这是虚拟机中用于挂载主机共享文件夹的目标目录。/mnt/hgfs 是一个常见的挂载点，用于存放主机共享的文件和文件夹。

bash
vmhgfs-fuse .host:/ /mnt/hgfs

umount -l /mnt/hgfs 强制取消挂载

ps aux | grep hgfs 查看占用挂载点的进程
